# Contributing to GEO Taxonomy

Contributions are welcome. This taxonomy improves when practitioners propose terms grounded in real GEO practice.

## How to Contribute

**Propose a new term:**
Open an issue with the proposed term, its category, a clear definition, and an example of usage in context.

**Submit a correction:**
Open a pull request with the fix. If it's a factual or definitional correction, include a source.

**Add a new export format:**
If you need the taxonomy in a format not yet provided (YAML, SKOS, etc.), submit a pull request with the converter and output file.

## Guidelines

- Definitions must be precise and falsifiable. Avoid vague or aspirational language.
- Each term must belong to exactly one category in the taxonomy.
- Provide evidence when possible. Link to documentation, research, or case studies.
- Write in English for global accessibility.
- Update all export formats (JSON, CSV, Markdown) when adding or modifying terms.

## Code of Conduct

Be professional and constructive. We're here to build a useful resource for the GEO community.
